Horiseon

Wedsite Design

In this app, the focus was on ensuring people with disabilities can access the Horiseon website using assistive technologies like video captions, screen readers, and a special keyboard. Making this website accessible may also help position it better in Google search engines .

In hopes of making my clients approving of my services, I also improved the codebase and reworked their previous file structure to make it more efficient, including comments before each element or section of the page.

Run Buddy

Wedsite Design

Working with the Run Buddy team was an exciting; making the project fun to work on. I was albe to stretch my knowledge of HTML and CSS. The Run Buddy had very specific criteria for this project. We added a form to the Reach Out section, so that users can submit questions directly to the sales staff. I also made updates for Hero Section to include inspirational message to connect with the community.

This project is version 2.0 for this company and includes some subtle visual changes like rounded corners on the forms. A few other changes reveal themselves as the user interacts with the page. The customer seemd please with the work and this has been the main focus all along.
